# Supply Chain Analytics Dashboard

This project is an interactive Supply Chain Analytics Dashboard built using Python and Microsoft Power BI to analyze operational performance, logistics efficiency, supplier quality, and inventory trends. The dashboard helps identify key business insights from supply chain data through clear visualizations and KPI tracking.

## Features

* KPI cards for Total Revenue, Average Shipping Cost, Defect Rate, and Inventory Levels
* Revenue analysis by product category
* Shipping cost comparison across transportation modes
* Supplier performance analysis using defect rates
* Manufacturing cost vs revenue scatter analysis
* Interactive slicers/filters for product type, supplier name, and transportation mode

## Tools & Technologies

* Python
* Pandas
* Microsoft Power BI
* CSV Dataset

## Key Insights

* Identified transportation modes with highest shipping costs
* Analyzed supplier defect trends and operational risks
* Compared manufacturing costs against generated revenue
* Evaluated inventory and product performance patterns

## Project Goal

The goal of this project is to demonstrate data cleaning, exploratory data analysis, business intelligence visualization, and dashboard storytelling skills using real-world supply chain data.

This project is suitable for showcasing data analytics, dashboard design, and business insight generation skills for internships and entry-level data analyst roles.
